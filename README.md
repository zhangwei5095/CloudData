CloudData
========

  这是一个云端的多租户平台，我们希望通过最简单的配置方式实现中小企业对于数据的管理。
  
  这不是一个追逐新技术的平台，我们崇尚通过最合理的方式解决技术难题。
  
  这是一个漫长而持久的计划，我们想要打造mini版的salesforce。
  
  如果你现在的工作满足不了你玩代码的激情，我们可以一起玩。
  
  如果你希望通过交流产品设计的心得获得更多的朋友，我们可以一起玩。
  
  如果你觉得你的web设计才是最好的，那么，期待你的建议。
